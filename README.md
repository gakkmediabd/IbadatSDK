## Setup

The simplest way to use SweetAlertDialog is to add the library as aar dependency to your build.

[Main/Parent Project Url](https://github.com/)

**Maven**
<dependency>
<groupId>com.ibadat.sdk</groupId>
<artifactId>library</artifactId>
<version>1.0</version>
<type>aar</type>
</dependency>

**Gradle**

    repositories {
        mavenCentral()
    }

    dependencies {
       
    }
